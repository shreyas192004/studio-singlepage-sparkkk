# Project

This repository contains the TESORA single-page storefront built with React, TypeScript, Vite, Tailwind CSS and shadcn-ui.

## Developing locally

Prerequisites: Node.js and npm (or pnpm/yarn).

Quick start:

```sh
git clone <YOUR_REPO_URL>
cd <YOUR_PROJECT_DIR>
npm install
npm run dev
```

Open `http://localhost:5173` (or the port shown) to preview the site.

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui

## Deployment

Build for production and deploy the generated `dist/` folder to your host of choice:

```sh
npm run build
npm run preview
```
